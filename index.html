Ejercicio 1


function suma(a,b,callback){
let c= a+b;
callback();
}
El tercer parametro recibido es callback, es un llamado a otra function. Es decir se pasa una funcion como argumento de otra funcion, la cual se ejecuta despúes que se haya completado la funcion principal (en este caso suma()). Es obligatorio desarrollar una funcion callback, cuando quiero ejecutar una funcion externa dentro de otra function.
